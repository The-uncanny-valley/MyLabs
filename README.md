## Структура репозитория

Каждая лабораторная работа находится в отдельной папке.

## Как запустить

1. Перейти в нужную папку:
```bash
   cd lab23
```
Скомпилировать:
```bash
   g++ main.cpp -o lab23
```
Запустить:
```bash
   ./lab23
```
## Требования
- C++11 или новее
- g++, clang++ или любой другой совместимый компилятор
